# PROBLEM OF THE DAY : Power of Two

## STATEMENT 
Given an integer n, return true if it is a power of two. Otherwise, return false.<br>
An integer n is a power of two, if there exists an integer x such that n == 2^x.
## Constraints

-2^31 <= n <= 2^31 - 1
